# Тестовое задание "Менеджер учёта расходов"
НАСТРОЙКА
В settings.py настроить следующие поля, в соответствие с выбранной почтой для отправки писем:

EMAIL_HOST = 'smtp.mail.ru'
EMAIL_PORT = 2525
EMAIL_HOST_USER = 'secret@mail.ru'
EMAIL_HOST_PASSWORD = 'secret'
EMAIL_USE_TLS = True
EMAIL_USE_SSL = False

Проверить, что настройки ваше почты позволяют отправлять сообщения со сторонних приложений.

Создать файл .env .
Добавить данные переменных окружения.
Пример:

DB_ENGINE=django.db.backends.postgresql
DB_NAME=test
DB_USER=django_user
DB_PASS=secret
DB_HOST=db
DB_PORT=5432

Дальше можно выполнит docker-compose build, docker-compose up


О ПРИЛОЖЕНИИ

Регистрация и авторизация пользователя была выполнена с использованием djoser. urls в соответствии с документацией djoser.
Статистика пользователя находиться по url:
-(user/) - информация о текущем балансе и о сумме всех транзакций сгруппированных по категориям.
Транзакции пользователя CRUD:
-( transaction_list/) – представлен список всех транзакций с возможностью фильтрацией по времени, сумме, дате. Добавление новой транзакции
-( transaction_list/<pk>/) – информация о конкретной транзакции с возможность редактирования как всего объекта так и части параметров, есть возможность удалить объект.
Категории пользователя CRUD:
- (category_list/) – представлены все пользовательские категории. Добавление новой категории
- (category_list/<pk>/) – информация о конкретной категории, изменение или удаление.
Отправка писем по расписанию реализована Celery. Статистика отражает информацию о текущем балансе и транзакциях, упорядоченных по категориям, за последний день.
